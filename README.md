## Initial setup

Create a Python virtual environment and activate it:

```sh
% python3 -m venv env/
% source env/bin/activate
```

Install the requirements with `pip`:

```sh
(env) % pip3 install -r requirements.txt
```

You can deactivate the virtual environment by issuing:

```sh
(env) % deactivate
```
